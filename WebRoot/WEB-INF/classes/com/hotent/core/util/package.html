系统工具类包。

  • AppUtil 注入WEB的servletcontext和Spring 的ApplicationContext对象实例，并且可以取得ApplicationContext，servletcontext。
  • BeanUtils 反射类的工具类。
  • ConfigUtil 系统配置读取工具类。
  • ContextUtil 上下文对象获取类，用于读取当前的登录用户。
  • Dom4jUtil xml操作类。
  • FileUtil 文件操作工具类。
  • ImageUtil 图形文件操作工具类。
  • MapUtil hashmap取值工具类。
  • ResourceUtil 属性读取工具类。
  • StringUtil 字符串操作工具类。
  • SystemInfo 系统信息工具类。
  • TimeUtil 时间操作工具类。
  • UniqueIdUtil 唯一id工具类。
  • ZipUtil 压缩解压缩文件 工具类。
